# Apache Hadoop installation with Ambari

<br/>

### On Host Machine

    $ sudo vi /etc/hosts

```
#---------------------------------------------------------------------
# Ambari cluster
#---------------------------------------------------------------------

192.168.0.10 master.ambari master
192.168.0.11 node1.ambari node1
192.168.0.12 node2.ambari node2
192.168.0.13 node3.ambari node3

<br/>

    $ vagrant plugin install vagrant-hostmanager


<br/>

**vagrant scripts:**
https://github.com/matematika-org/vagrant-hadoop-ambari


Steps are the same as in cloudera instruction.


<br/>

http://master.ambari:7180/
```
